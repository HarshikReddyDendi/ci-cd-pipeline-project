# CI/CD Pipeline Automation Project

This project demonstrates a simple Continuous Integration (CI) pipeline using GitHub Actions.

## Technologies Used

* Python
* Git
* GitHub Actions

## Workflow

1. Code is pushed to the repository.
2. GitHub Actions automatically triggers the CI pipeline.
3. The pipeline sets up Python and runs the script.
